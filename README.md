# بورتفوليو الجوهرة الموسلم - Aljohara Almusallam Portfolio

موقع بورتفوليو احترافي لعرض المشاريع والخبرات في مجال الهندسة البيوميديكالية وتقنيات الرعاية الصحية.

## 🎯 محتوى الموقع

### الصفحات الرئيسية
- **الرئيسية (Home)** - صفحة استقبال بمعلومات عامة
- **التعريف الشخصي (Profile)** - نبذة عن الخبرة والمهارات
- **المشاريع (Projects)** - عرض المشاريع الأكاديمية والعملية
- **التدريب (Training)** - تفاصيل الخبرة التدريبية
- **الشهادات (Certifications)** - الشهادات والتدريبات المهنية
- **التواصل (Contact)** - معلومات التواصل

## 📋 المشاريع المعروضة

### مشروع التخرج
- **تطبيق الرعاية الصحية المنزلية** - منصة رقمية لحجز خدمات الرعاية الصحية

### المشاريع المختارة
- نظام كشف تفاعلات الأدوية
- نظام قياس مخطط القلب باستخدام Arduino
- تصميم كيس أمبو (3D Modeling)
- حل ضعف العضلات عند رواد الفضاء

## 🎓 التدريب والشهادات

### التدريب العملي
- مدينة الملك عبدالعزيز الطبية - الشؤون الصحية بوزارة الحرس الوطني

### الشهادات المهنية
- مهارات الذكاء الاصطناعي (AI Skills 4 Women)
- أساسيات الذكاء الاصطناع�� بـ Microsoft Azure
- برمجة الشبكات والأمان السيبراني
- MATLAB و معالجة الصور
- وغيرها...
